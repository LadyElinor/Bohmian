# Session Notes Archive

This folder holds dated working notes moved out of repo root to reduce orientation friction.

## Structure
- `2026-03/` — March 2026 session notes, sweeps, closure notes, and dated governance drafts.

## Conventions
- Keep canonical status/governance docs in root (e.g., `CLAIM_STATUS_MATRIX.md`, `GR_QM_CONSECUTIVE_CYCLE_PROMOTION_LEDGER.md`, `GR_QM_CYCLE_JOURNAL.md`).
- Put dated narrative notes and interim packets under `archive/session-notes/YYYY-MM/`.
- If a note is later promoted to canonical, copy or summarize it into the relevant canonical file rather than moving canonical docs back to root.
